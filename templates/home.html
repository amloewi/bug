bug

An insistent text message reminder service

bug reminds you of something, repeatedly, until you do it. Like a person would. Like you wish you could do for yourself. Because you know that's how things get done. It's that simple.

Using bug is just as simple. Here's what would happen if you sent it some texts.

Examples

Buy groceries for tomorrow!
If you sent bug this text, it would be sent back to you every 30 minutes, 5 times.

Return the library book 20
If you sent this one, the messages would be 20 minutes apart, but there'd still be 5.

Go to the gym before dinner 60 2
You'd get this message 60 minutes apart, 2 times.

Did beetle/ Cancel ant
When you set a reminder, bug assigns it a name, like 'beetle', and sends you a message to tell you what that name is. You can use that name to stop the reminder, if you finish the thing you wanted to do (congratulations!) or cancel it, if you made a mistake.


If you have a number at the end of your message like "Dinner at 8", just make sure it has a period (or something) after it to make it clear it's not meant to be a parameter.

bug will do the rest.
