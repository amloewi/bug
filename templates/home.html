bug

An insistent text message reminder service

Alarms only go off once. When they do go off, you're busy, and an hour after the alarm, the thing you needed to do still needs to be done, but you've forgotten about it again. Especially if you're sorta avoiding it on purpose ... but you can't let me keep doing that!

bug takes the more intuitive, and direct approach. It reminds you of something, repeatedly, until you do it. Like a person would. Like you wish you could do for yourself. Because you know that's how things get done.

The interface is just as simple. Send a text to the bug number in the format:

name, message [, interval=30 [, repeats=5 ]]


Where name is the name of the reminder (which in the future, you'll be able to use to cancel the reminders once you've done the thing), message is the message you'll get when the reminder is sent, interval is the number of minutes in between reminders (which defaults to 30, if you don't enter anything), and repeats is the number of times the reminder will be sent (which defaults to 5, if you don't enter anything).

Examples

Groceries, Buy groceries for tomorrow!, 60, 3
This will send the message 3 times, 60 minutes apart.

Book, Return the library book, 20
This will send the message 5 times (because that's the "repeats" default), 20 minutes apart

Gym, Go the gym before dinner at 8
This will send the message 5 times (by default) 30 minutes apart (by default).

bug is in the middle of development, so things are going to change, and also not always work. In particular, at the moment, it's finicky about input format, so be careful -- and don't put commas in your message!
